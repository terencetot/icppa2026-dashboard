# ICPPA 2026 — Abstract Review Dashboard

Interactive dashboard for the ICPPA 2026 abstract scoring process.

**Live dashboard:** https://terencetot.github.io/icppa2026-dashboard

## Features
- Abstract rankings by mean score
- Standardized Z-scores
- Reviewer completion tracking
- Incomplete review tracking with priority levels
- Filters by gender, language, review count, score threshold
